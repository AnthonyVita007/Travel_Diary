# Progetto Mobile Programming - Gruppo 15
Travel Diary App
L’obiettivo è sviluppare un’app multipiattaforma per registrare i propri viaggi. Gli utenti possono creare “entry” di viaggio, organizzare le destinazioni e visualizzare statistiche sulle esperienze fatte.
## 🎯 Cosa può fare l'app (Features Implemented)

### ✨ Gestione Viaggi
- **Visualizza tutti i viaggi** in una lista scrollabile con immagini e dettagli
- **Crea nuovi viaggi** con titolo, immagine, date, descrizione e categoria
- **Modifica viaggi esistenti** con editor completo
- **Elimina viaggi** con conferma di sicurezza
- **Visualizza dettagli completi** di ogni viaggio

### 🔍 Ricerca e Filtri
- **Ricerca in tempo reale** per titolo, descrizione o categoria
- **Filtra per categoria** (Nature, Safari, Off Road, City Life, Beach, Mountain, Culture, Food & Wine)
- **Filtra per preferiti** per vedere solo i viaggi favoriti
- **Indicatori di filtro attivi** con reset rapido

### ❤️ Sistema Preferiti
- **Aggiungi/rimuovi dai preferiti** con pulsante cuore interattivo
- **Indicatori visivi** sui viaggi preferiti (stella dorata)
- **Filtro dedicato** per visualizzare solo i preferiti

### ⚙️ Impostazioni
- **Informazioni sull'app** e gruppo di sviluppo
- **Gestione notifiche** e modalità scura (toggle)
- **Backup dati** e pulizia dati
- **Statistiche personali** (viaggi totali, preferiti, categorie)
- **Supporto e aiuto**

### 🎨 Interfaccia e Navigazione
- **Design moderno** con Material Design icons
- **Navigazione fluida** tra schermate
- **Barra di navigazione inferiore** per accesso rapido
- **Gestione stati vuoti** con messaggi informativi
- **Layout responsive** adattabile

### 📊 Categorizzazione
- **8 categorie predefinite** con icone e colori personalizzati
- **Selezione visuale** delle categorie durante creazione/modifica
- **Filtro per categoria** nella ricerca

## 🏗️ Architettura
- **React Native** con Expo
- **React Navigation** per la navigazione
- **Modello dati strutturato** con classi Trip e TripCollector
- **Gestione stato locale** con React Hooks
- **Componenti riutilizzabili** per UI consistente

## 🚀 Come avviare l'app
```bash
npm install
npm start
```
